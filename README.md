# Xode Blockchain

## Overview

Xode is a blockchain platform built on Substrate, designed for [project description].

## Features

- Substrate-based blockchain
- Smart contract support
- Cross-chain interoperability
- High performance consensus

## Quick Start

### Prerequisites

- Rust toolchain
- Node.js 18+
- Docker (optional)

### Installation

```bash
# Clone the repository
git clone https://github.com/Xode-DAO/xode-blockchain.git
cd xode-blockchain

# Build the project
cargo build --release

# Run tests
cargo test
```
